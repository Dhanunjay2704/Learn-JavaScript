# Learn JavaScript - Complete Learning Journey

A comprehensive, phase-based JavaScript learning repository covering fundamentals to advanced concepts with practical exercises and projects.

## üìö Project Structure

This repository is organized into **6 phases**, progressively building JavaScript skills from basics to advanced topics.

---

## Phase 1: JavaScript Fundamentals

**Topics Covered:**
- `01_JS_VARIABLE/` - Variable declaration and scope
- `02_DOM/` - Document Object Model basics
- `03_EVENTS/` - Event handling
- `04_DATA_TYPES/` - Primitive and complex data types
- `05_CONDITIONAL_STATEMENTS/` - If/else, switch statements
- `06_RANDOM_NUMBERS/` - Random number generation
- `07_HTML_INPUT/` - Working with HTML input elements
- `FINAL_CODE/` - Phase 1 capstone project

---

## Phase 2: Arrays, Functions & Objects

**Topics Covered:**
- `01_ARRAYS/` - Array creation, manipulation, and methods
- `02_FUNCTIONS/` - Function declaration, expressions, and arrow functions
- `03_DOM_MANIPULATIONS/` - Advanced DOM manipulation techniques
- `04_OBJECTS/` - Object creation, properties, and methods
- `FINAL_CODE/` - Phase 2 capstone project

---

## Phase 3: Storage & Advanced Array Methods

**Topics Covered:**
- `01_MORE_HTML_ELEMENTS/` - Working with various HTML elements
- `02_STORAGE_MECHANISM/` - localStorage and sessionStorage
- `03_NULL_VALUES_AND_TEXTAREA/` - Handling null values and textarea elements
- `04_JSON_NOTATION/` - JSON parsing and stringification
- `05_ARRAY_METHODS/` - Array methods (map, filter, reduce)
- `06_ARRAY_METHODS_PART2/` - Advanced array methods
- `07_FINAL_CODE/` - Phase 3 capstone project

---

## Phase 4: Events & Advanced Interactions

**Topics Covered:**
- `01_CALLBACK_FUNCTION/` - Callback functions and patterns
- `02_SCHEDULERS/` - setTimeout and setInterval
- `03_EVENT_LISTENERS/` - Event listener patterns
- `04_OPERATORS/` - Logical, comparison, and special operators
- `05_KEYBOARD_EVENTS/` - Keyboard event handling
- `06_EVENT_OBJECT/` - Understanding the event object
- `FINAL_CODE/` - Phase 4 capstone project

---

## Phase 5: HTTP & Fetch API

**Topics Covered:**
- `01_HTTP/` - HTTP fundamentals and XMLHttpRequest
- `02_HTTP_PART_2/` - Advanced HTTP concepts
- `03_HTTP_FETCH/` - Fetch API and modern async patterns
- `FINAL_CODE/` - Phase 5 capstone project

---

## Phase 6: Forms & Final Integration

**Topics Covered:**
- `01_FORMS/` - Form handling and validation
- `FINAL_CODE/` - Phase 6 capstone project

---

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dhanunjay2704/Learn-JavaScript.git
   ```

2. **Navigate to desired phase:**
   ```bash
   cd "PHASE 1"
   ```

3. **Open files in your editor:**
   - Each lesson has an `index.html` file (with DOM/UI) and `index.js` file (logic)
